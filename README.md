# 🚨 Fiscal de Procrastinação (Chrome Extension)

Extensão nativa para Google Chrome focada em produtividade. Ela bloqueia o acesso a sites distrativos com uma interface de alerta e permite gerenciar a lista de bloqueio em tempo real.

## 🚀 Tecnologias Utilizadas
- **JavaScript (Vanilla)**
- **Chrome Extensions API (Manifest V3)**
- **Chrome Storage API:** Para salvar a lista de sites bloqueados.

## ⚙️ Como instalar
1. Acesse `chrome://extensions/` no seu navegador.
2. Ative o **"Modo do desenvolvedor"**.
3. Clique em **"Carregar sem compactação"** e selecione a pasta deste projeto.
